# 突围俱乐部官网

> 13 年坚持，一群人同行。突围，不只是俱乐部，更是一个家。

## 项目信息

- **成立时间**: 2012 年
- **活动场次**: 660+ 场
- **参与人次**: 11500+
- **会员人数**: 5000+

## 核心活动

- 🏃 **户外活动** - 夜登南山（334 期+）、七娘山、马峦山、麦理浩径
- 📚 **读书会** - 深度阅读，思维碰撞
- 💼 **创业赋能** - AI 沙龙、创业分享、资源对接
- 🤝 **社交连接** - 找到志同道合的朋友

## 技术栈

- **前端**: Next.js 14 + React 18 + Tailwind CSS
- **部署**: Vercel
- **数据库**: PostgreSQL
- **存储**: 阿里云 OSS

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 启动生产服务器
npm start
```

## 项目结构

```
towayclub-website/
├── app/
│   ├── layout.js      # 根布局（导航 + 页脚）
│   ├── page.js        # 首页
│   ├── events/        # 活动页面
│   ├── outdoor/       # 户外页面
│   ├── reading/       # 读书页面
│   ├── startup/       # 创业页面
│   ├── membership/    # 会员页面
│   └── about/         # 关于页面
├── components/         # 可复用组件
├── public/            # 静态资源
└── package.json
```
